# BME I2C Pico W wiring
- [Adafruit BME280](https://www.adafruit.com/product/2652) sensor uses address 0x77
- Adafruit BME280 sensor is configured for I2C connection
| BME280 Pin | Pico W Pin |
|------------|------------|
|   VIN      | 3.3V (OUT) |
|   GND      | GND        |
|   SCK      | GPIO 5     |
|   SDI      | GPIO 4     |
