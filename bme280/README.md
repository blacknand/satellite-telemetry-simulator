# BME I2C Pico W wiring
- [Adafruit BME280](https://www.adafruit.com/product/2652) sensor uses address 0x77

| BME 280 pin | Pico W pin |
| ------- | ------ |
| VCC   | 3.3V (OUT) | 
| GND        | GND |
| SCK        | GPIO 5 |
| SDI | GPIO 4 |

# MPU6050 Pico W sensor wiring
| MPU6050 pin | Pico W pin |
| ------- | ------ |
| VCC   | 3.3V (pin 36) | 
| GND        | GND (pin 2) |
| SCK        | GPIO 0 |
| SDI | GPIO 1 |
